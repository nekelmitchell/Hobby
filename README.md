# Hobby Hop Game🐰

Hobby Hop is a beginner single-player 2D arcade game inspired by Flappy Bird
The player controls a rabbit named Hobby and must avoid touching poisonous carrots while staying alive as long as possible.

## Gameplay

* Press the **Space Bar** to make Hobby hop.
* Avoid the poison carrots.
* Try to see how long you can survive for.

## Features

* Physics-based jumping mechanics
* Obstacle spawning system
* Collision detection
* Game over and restart functionality
* Simple 2D gameplay design

## Built With

* Unity
* C#
